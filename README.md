networq:core
====

This is the "networq:core" package for [NetworQ](https://github.com/networq).

## Types

This package defines the following types that you can use in your own Networq:

### networq:core:node type

Fields:

  * `name`: *string*
  * `description`: *string*
  * `image`: *string*
  * `url`: *string*
  * `details`: *string*
